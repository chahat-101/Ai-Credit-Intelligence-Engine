- Python
- Basic understanding of machine learning concepts

- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Git & GitHub
- Forking repositories
- Creating branches
- Making pull requests

## General
- Problem solving
- Reading existing code
- Willingness to learn

## Optional
- Data preprocessing
- UI improvements in Streamlit
- Model tuning
